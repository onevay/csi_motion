# Notebooks: что здесь лежит и зачем

Это «витрина» проекта: картинки, графики обучения и короткие пояснения, чтобы репозиторий можно было открыть и быстро понять смысл, не читая весь код.

Картинки лежат рядом, в `notebooks/figures/`.

## 1. Цель

Задача: по CSI с трёх приёмников определить движение и расстояние (0/1/2/3 м). Здесь фиксируются метрики и визуализации, чтобы было видно, как ведут себя модели на полном наборе и на самом сложном сценарии (3-й человек в LOPO).

## 2. Данные (коротко)

- Окно: 100 временных точек × 52 поднесущих × 3 приёмника.
- Разметка расстояния: 4 класса. Для multitask-моделей движение — это «есть ли активность», то есть `label > 0`.
- Валидация: Leave-One-Person-Out (LOPO), в таблице ниже два числа: полный набор / удержанный человек.

## 3. Препроцессинг (как это выглядит снаружи)

На уровне репозитория цепочка такая:

- `tools/receiver.py` складывает синхронные сегменты в `test_*/dev{1..3}.data`.
- `cpp_preprocessor/build/preprocessor` читает три файла, достаёт амплитуды, прогоняет медианный фильтр по времени и печатает три уплощённых массива.
- `tools/watcher.py` (или `main.py watcher`) превращает это в тензор `(100, 52, 3)` и гоняет модели.

Если хочешь добавить ещё графики «до/после фильтра» или распределения шумов, просто клади файлы в `figures/` и вставляй ссылку ниже по аналогии.

## 5. Модели (чем они отличаются по смыслу)

### CNN+LSTM+Attention

Классический путь: свёртки вытаскивают локальные паттерны по времени, LSTM держит длинную динамику, attention выбирает «важные» моменты окна. Две головы: motion и distance.

### Mamba-hybrid

Более лёгкая по духу ветка для длинных рядов: локальные свёрточные блоки + state-space часть, плюс взаимное внимание между представлениями для motion/distance.

### Transformer fusion

Отдельная линия: внутри строятся дифф-признаки по времени, есть две ветки и обучаемое смешивание. На входе для этой модели используется нормализация по `train_mean.npy` / `train_std.npy`.

## 7. Выводы

Если коротко: CNN+LSTM выглядит сильнее по distance на сложном человеке, Mamba держит motion очень уверенно, а Transformer-ветка в этом репозитории заточена под distance и живёт своей нормализацией.
